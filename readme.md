# Dictionary.

## 一个易于扩展的翻译程序。

基本:
> Python3.4+ <br />
> PyQt55.5+ <br />
> requests <br />

划词方面:
> pywin32 <br />
> keyboard <br />
> pywinauto <br />

> 划词使用 ctrl + c 的复制方法划词。监视鼠标会导致鼠标有些许卡顿。
